# WHY LANGCHAIN?
# --------------
# LangChain provides standardized interfaces for:
#   - LLMs (OpenAI, Google, Anthropic, local models)
#   - Embeddings (same providers)
#   - Vector stores (Chroma, Pinecone, Weaviate, etc.)
#   - Chains (composable pipelines)
#   - AGENTS (ReAct, function calling, tool use)
#
# This means you can swap providers without rewriting your code!
#
# AGENT VS WORKFLOW:
# ------------------
# This project uses a ReAct AGENT, not a fixed workflow:
# - Agent REASONS about what to do next
# - Agent DECIDES which tools to use
# - Agent ADAPTS based on intermediate results
# - Agent SELF-CORRECTS when validation fails
#
# LANGCHAIN ECOSYSTEM:
# --------------------
# - langchain-core: Base abstractions and LCEL
# - langchain: Agents, chains, output parsers
# - langchain-google-genai: Google/Gemini integration
# - langchain-chroma: ChromaDB vector store
#
# =============================================================================
